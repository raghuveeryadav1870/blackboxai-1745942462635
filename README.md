# Netflix Clone

Welcome to the Netflix Clone project! This web application replicates the essential features of Netflix, allowing users to browse movies and TV shows, upload new content, and access an engaging user interface designed with a dark theme and smooth navigation.

## Project Overview

The main purpose of this project is to create a aesthetically pleasing and functional web application inspired by the Netflix streaming service. It includes sections for trending movies, user sign-ins, and an upload form for adding new movies to the platform. The application leverages HTML, CSS (with Tailwind CSS for styling), and JavaScript for interactivity.

## Installation

To run this project locally, follow these steps:

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```
   
2. Navigate to the project directory:
   ```bash
   cd netflix-clone
   ```
   
3. Open `index.html` in your preferred web browser.

## Usage

1. Open the application in your web browser.
2. Browse the sections for trending movies and the upload form.
3. To upload a movie, fill out the form with the required details and click the "Upload Movie" button.
4. The uploaded movie will be displayed in the "Trending Now" section.

## Features

- **Responsive Design**: Adaptable layout for various devices including desktop, tablets, and smartphones.
- **Navigation Menu**: Simple and user-friendly navigation for easy access to different sections.
- **Movie Upload Form**: Allows users to upload new movies with title, genre, poster URL, and description.
- **Trending Movies Section**: Displays currently trending movies with an interactive layout.
- **Dynamic Messages**: Feedback messages displayed upon successful movie uploads or error notifications.

## Dependencies

This project utilizes the following external libraries and frameworks:

- [Tailwind CSS](https://tailwindcss.com/) - Utility-first CSS framework for styling.
- [Font Awesome](https://fontawesome.com/) - Provides icons for the UI.
